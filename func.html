Functions

In this section, we will learn about functions and their cool shit!.

What the hell is a function?!

A function is a bunch of code you can reuse as needed. They are like mini programs in your main program, that can be used or called to do some shit when you need it! They can also help organize your code to make it easier to understand.

How to make functions

To make a function, use the def word, meaning define. Type in the name of the function, which can be almost anything, like tuba, or angryGrandpa, or even ass! Now put a pair of parentheses right next to your function name, with no spaces. Now put a colon (:) next to the parentheses. It will look something like this: 


def angryGrandpa():
Now press Enter/Return and see what happens. It goes to the next line and indents for you! Indentation is important in Python. Anything that is indented is part of the function, whereas anything else is not part of the function. Whatever is in the function, via the indentation will be used when you call the function. Here, type this in the window then run the code when you finished typing it: 

#!/usr/bin/env python3
"""
This is a goddamn function!
About Angry Grandpa!
"""

def angryGrandpa():
	print("Holy motherfucking shit!")

angryGrandpa()
It should print this: Holy motherfucking shit! What it did is that it executed the function angryGrandpa(), which prints out "Holy motherfucking shit". We called the function so we could use it by simply typing the function name with the parentheses right after the name. Sweet shit!

More examples

Here is another example to help you understand functions better. 


#!/usr/bin/env python3
# This will use a function that will print a string and a number.
tuba = 10000

def doItAgain():
	print("The tuba is loud!")
	print(tuba)

doItAgain()
This is what the program will do: 

The tuba is loud!
10000

Conclusion

You learned about functions and their sweet usefulness! They can be used to tidy up your code and are like mini programs that can be used over and over!
