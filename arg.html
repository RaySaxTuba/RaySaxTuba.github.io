Parameters and Arguments

In this section, we will learn about parameters and arguments of functions.

WTH are parameters and arguments?!

Parameters are additional properties of functions. Arguments are values of the parameters. For example, let's write this: 


#!/usr/bin/env python3
"""
This function has the parameter called name and the parameter's argument
is a string called Buttnuggets.
"""

def greet(name):
	print("Hello " + name + "!")

greet("Buttnuggets")
The following code will spit out: Hello Buttnuggets! We made a function called greet. Its parameter, name was assigned a string called "Buttnuggets", which is the argument. Wow!

Default arguments of parameters

Let's rewrite the above example code to demonstrate default arguments: 


#!/usr/bin/env python3
"""
This function has the parameter called name and the parameter's default
argument is a string called Angry Grandpa.
"""

def greet(name = "Angry Grandpa"):
	print("Hello " + name + "!")

greet()
greet("Asstuba")
The following code will spit out: 

Hello Angry Grandpa!
Hello Asstuba!
So if you don't supply any arguments to the greet function, it will use the default argument, "Angry Grandpa". But if you do supply an argument like in the above code, "Asstuba", it will use that instead!

More examples

Here is another example to help you understand functions better. 


#!/usr/bin/env python3
# This will use a function that will print a string and a number.
tuba = 10000

def doItAgain():
	print("The tuba is loud!")
	print(tuba)

doItAgain()
This is what the program will do: 

The tuba is loud!
10000

Conclusion

You learned about functions and their sweet usefulness! They can be used to tidy up your code and are like mini programs that can be used over and over!
