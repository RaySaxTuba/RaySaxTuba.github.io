Strings in programming

In this section, we will learn about strings!

WTF is a string?

A string is a bunch of letters, numbers, and/or symbols that are joined together with quotation marks like this: "The force is strong with mah stinky breath!" or 'Mah candy!'. It's pretty much text! It's a string of text! Remember in the first section where we made our first program? We actually outputted a string that was "Mah candy!".

How to make a string

To make a string, we use quotation marks ("") or ('') and type in shit inside of them! We can also store strings in variables! Here's an example: 


name = "Angry Grandpa Tuba!"
The string "Angry Grandpa Tuba!" got stored in the variable name We can print the string by using print(name). 

name = "Angry Grandpa Tuba!"
print(name)
We printed out the name of Angry Grandpa!

Concatenation? WTF is that?!

Alright! Let's learn about concatenation and what it has to do with strings. Concatenation is pretty much linking strings together or even with variables! Here's an example: 


name = "Angry Grandpa Tuba! "
name2 = "He's the shit!"
print(name+name2)
And we frigging get: Angry Grandpa Tuba! He's the shit! We use the + operator to link, or concatenate the two strings! Very frigging cool!

Escape characters

Sometimes you may type a string in the manner of this: 


quote = "Angry Grandpa says, "Mah candy!""
It looks fucked up in the window! Goddamn! To fix this, we can use single quotation marks, like: 

quote = 'Angry Grandpa says, "Mah candy!"'
Or we can use something called an escape character. An escape character is a backslash (\) that we can use on the quotes inside of the string to make Python go like, OK, these can be here! Jeez! Let's type the above code with the escape characters: 

quote = "Angry Grandpa says, \"Mah candy!\""
Pretty much, it allows us to use the double quotation marks inside the string without fucking up anything. Sweet tuba!

Other escape tricks

You can make some of the string go to another line, like this: 


quote = "The force \n
is strong with \n
mah stinky breath!"
It will print like this: 

The force
is strong with
mah stinky breath!

Turning numbers into strings and back!/Numberbots, transform and roll out!

We can also turn numbers into strings, like they are Transformers! Here's an example: 


age = 4
print("Mah age is" + str(age))
It will say, "Mah age is 4". Note: if we did not put the str() function around the age variable, we would get an error, because we cannot concatenate strings and numbers together. It just doesn't frigging work! So we have to turn the number into string format so no errors come up. We can also turn a string with a number in it into a pure number as well. For example: 

int("4")
This will turn the string with 4 in it, into the actual number 4!

Conclusion

You learned about strings and their awesome shit! They are pretty much text! Tuba!
