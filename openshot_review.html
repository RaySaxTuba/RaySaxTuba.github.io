Review of Openshot

What is this program?

Openshot is an open source video editor that has functionality similar to commercial video editors like Final Cut Pro, Premiere, and Media Composer. Openshot was created in 2008 by Jonathan Thomas.

Main review

I think this program has the potential to be the definitive open source video editor, but unfortunately falls short by quite a bit. The features of this program include:

  • Support for most video files for use in the user's project.
  • Importing SVG files to use as titles.
  • Exporting to a couple of video formats.
  • Compatibility with Linux, Windows, and Mac.
  • Ability to make animated titles via Blender.
  • Effects like slow motion, fade in and out, and more.
So where does this program fall short? It crashes easily. Oh snap! It crashes at a random time or usually from putting a time effect on a video. This program also has a complicated time trying to do animated titles.i7
