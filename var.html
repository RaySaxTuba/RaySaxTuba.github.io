Variables in programming

In this section, we will learn about variables!

What the hell is a variable? In programming?

A variable in programming is kind of like a variable in math. It is a placeholder for something else, like a number. Variables in programming can hold anything, like numbers, strings (more on strings later), functions, and other things!

How to make a variable

To make a variable, we first need to know the proper way to do it. Here is a list on what to do and what not to do when making a variable:

  • Do's:
    • Make the variable a short word, like rage.
    • Start the variable with a lowercase letter.
    • Use camel case, like mahCandy, for separating the individual words in the variable name.
  • Don't's:
    • Use a number at the start of the variable name, like 9lives.
    • Use spaces, like mah candy. Use underscores instead of spaces, like mah_candy.
    • Use a math operator at the start of a variable name, like *tuba
    • Use a reserved word in Python, like for.
Now to actually make variables! Let's actually make a program that we can save on our computer! Click on File, click on New File, and then a blank window will pop up (You can also press Ctrl+N or Cmd+N). You can type code in this blank window and save it for later! In fact, I prefer you type in code like this unless I say otherwise. Type in this: 
          
            tuba = 10000
            print(tuba)
Then go to Run, click on Run Module, and see the shit happen in the other window! What happened is that we made a variable named tuba and printed out the value of the variable, 10000. You can change the variable name to something else like balls and change 10000 to something else as well!

Conclusion

You learned about variables in programming! Variables are very useful in programming because they can store anything in them and you can use the variable later on as well!
